Spatiotemporal UCNP Interactions

This repository contains Python scripts for analyzing the spatiotemporal interactions of various biological structures, including Lysosome, Mitochondria (Mito), Endoplasmic Reticulum (ER), and Golgi. The analysis focuses on understanding the positioning of these organelles and their interactions.

Files in this Repository
Lysosome position figure.py: Script for visualizing the positioning of lysosomes.
Lysosome position.py: Script for calculating and analyzing the positioning of lysosomes.
Mito ER Golgi proximity_probability_randomization_summary_threshold_robustness.py: A comprehensive script to analyze mitochondrial, ER, and Golgi proximity data.
Mito position figure.py: Script for visualizing the positioning of mitochondria.
Mito-position.py: Script for calculating and analyzing the positioning of mitochondria.
Golgi-position.py: Script for analyzing the positioning of the Golgi apparatus.
ER-position figure.py: Script for visualizing the positioning of the Endoplasmic Reticulum (ER).
ER_Position.py: Script for calculating and analyzing the positioning of the ER.
Installation

Clone this repository:

git clone https://github.com/Great-sophie/Spatiotemporal-UCNP-Interactions.git

Navigate to the directory:

cd Spatiotemporal-UCNP-Interactions

Install the required dependencies:

pip install -r requirements.txt
Usage

To run any of the Python scripts, use the following command:

python <script_name>.py

For example:

python Lysosome\ position.py

Each script contains its own set of parameters and instructions for how to use it. Make sure you review the code comments to understand how to adjust the settings for your own data.
